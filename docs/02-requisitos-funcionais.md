# CampoDay

# Requisitos Funcionais

Este documento descreve todas as funcionalidades que deverão ser implementadas no sistema CampoDay.

---

# Módulo 1 - Autenticação

## RF001
O sistema deve permitir que um usuário realize login utilizando e-mail e senha.

## RF002
O sistema deve permitir recuperação de senha.

## RF003
O sistema deve manter a sessão autenticada.

## RF004
O sistema deve permitir logout.

---

# Módulo 2 - Usuários

## RF005
O administrador deve poder cadastrar usuários.

## RF006
O administrador deve poder editar usuários.

## RF007
O administrador deve poder desativar usuários.

## RF008
O sistema deve possuir perfis de acesso.

Perfis:

- Administrador
- Coordenador
- Agrônomo
- Equipe de Apoio
