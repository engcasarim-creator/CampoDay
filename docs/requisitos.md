# Documento de Requisitos

## Projeto

CampoDay

## Objetivo

Desenvolver um sistema para gerenciamento de Dias de Campo promovidos por cooperativas agroindustriais.

---

## Perfis de Usuário

### Administrador

- Gerenciar usuários
- Visualizar eventos
- Emitir relatórios

### Agrônomo

- Criar eventos
- Organizar checklist
- Gerenciar materiais
- Registrar participantes
- Gerar relatório

---

## Funcionalidades do MVP

- Login
- Cadastro de eventos
- Checklist
- Materiais
- Participantes
- Check-in
- Fotos
- Relatórios
